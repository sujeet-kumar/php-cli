# php-cli
PHP library to build command line tools

-by Sujeet <sujeetkv90@gmail.com>

Install using composer:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sujeetkv/php-cli"
        }
    ],
    "require": {
        "sujeetkv/php-cli": "dev-master"
    }
}
```
