# php-cli
PHP library to build command line tools

-by Sujeet <sujeetkv90@gmail.com>

Install using composer:

    {
        "repositories": [
            {
                "type": "vcs",
                "url": "https://github.com/sujeet-kumar/php-cli"
            }
        ],
        "require": {
            "sujeet-kumar/php-cli": "dev-master"
        }
    }
